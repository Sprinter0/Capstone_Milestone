Dare Greatly

In All Star Code SI, we were encouraged to dare greatly by taking risks and pushing beyond our comfort zones, knowing that bold attempts lead to meaningful progress. In the All Star Code SI, we were encouraged to tell our stories, sharing the unique journeys and experiences that shaped our approach to programming. Takeaways

Week 1

As we dove into JavaScript fundamentals, we learned how to use variables, data types, and operators to build the foundation of our coding projects. Mastering control structures like loops and conditionals helped us create dynamic programs, while discovering functions and objects improved our ability to organize and manage complex information.

Week 2

We learned to use HTML for structuring and CSS for styling our web pages, allowing us to control content and design. For example, our week 2 portfolio was a major step up from anything we had done before.

Week 3

We used HTML, CSS, and JavaScript to put together a project with our teammates, using HTML to organize the content, CSS to make it look good, and JavaScript to add interactive features. This mix helped us create a project that was both fun and functional. Working together, we improved our skills in building complete and into a website .
